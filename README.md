Menggunakan Aplikasi VSCODE (Python) untuk coding dan Github untuk collab

ALUR 
1. Pengguna menjalankan program untuk membuat invoice.
2. Pengguna memasukkan data transaksi berupa tanggal transaksi, nomor invoice, nama pembeli dan alamat pembeli.
3. Pengguna memasukkan jumlah barang yang ingin dibeli lalu memasukkan data barang berupa nama barang dan kuantitas.
4. Sistem menentukan harga satuan berdasarkan nama barang yang dipilih lalu menghitung total harga setiap barang berdasarkan kuantitas dan harga satuan.
5. Sistem menghitung subtotal dari seluruh barang sebelum diskon dan pajak.
6. Jika subtotal lebih dari Rp 2.000.000, sistem memberikan diskon sebesar 10%.
7. Sistem menghitung pajak sebesar 5% dari subtotal setelah dikurangi diskon.
8. Sistem menghitung total tagihan akhir (Subtotal - Diskon + Pajak).
9. Sistem menampilkan dan mencetak invoice yang berisi detail transaksi dan total pembayaran.
